# Objective #3: Notification System and AI-Driven Prescriptive Recommendations

**Owner (Member Name):** Erick Layug

## Objective Title

Notification System and AI-Driven Prescriptive Recommendations

## Objective Description

This objective focuses on implementing automated notifications and AI-driven recommendation features. The system provides alerts for due and overdue payments, lease expirations, room availability, and maintenance requests while generating recommendations that help apartment owners improve revenue collection and room occupancy.
